# A&M Masonry Website

## Overview
The website was created for A&M Masonry Inc., a construction company based in Lawrenceville, Georgia. This website serves as a 
place to showcase their work, build a brand, and attract new clientele.

## Features
- Responsive homepage design
- Masonry services section
- Project gallery section
- About the company section
- Contact call-to-action
- Mobile-friendly layout

## Tech Stack
- HTML
- CSS
- JavaScript
